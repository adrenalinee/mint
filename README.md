# Mint
Mint 는 사용자가 다양한 용도로 기능을 확장하여 사용될 수 있는 angular 로 개발된 web Http Client component framework 입니다.

# Getting started
```
git clone https://github.com/adrenalinee/mint.git
cd mint
npm install
npm start
```
**웹 브라우저에서 http://localhost:3000 접속**
